# 🛒 Tapas Stores Web App

This is a simple Flask-based web application with login, register, and product management system.

---

## 🚀 Features

- User Registration
- User Login (Session based)
- Product listing
- Add products
- Logout system
- Simple frontend (HTML + CSS)

---

## 🧱 Tech Stack

- Python
- Flask
- HTML
- CSS
